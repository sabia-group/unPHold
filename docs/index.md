# unPHold

**unPHold**: Phonon band unfolding for supercells: relaxation-aware, layer-resolved, and MLIP-compatible.

## Features

- Unfold different materials: 3D, 2D, 1D, twisted bilayers, heterostructures, defects.
- Deregistration-aware: handles atomic displacements from perfect registry (e.g. strong buckling in 2D heterostructures).
- Compatible with MLIPs and DFT calculations.
- Phonon metrics: acoustic participation ratio (APR), longitudinality (L), and verticality (V).
- Phonopy-compatible: reads force constants and band structures directly, easy to extend.
